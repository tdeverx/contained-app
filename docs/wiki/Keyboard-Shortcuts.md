# Keyboard Shortcuts

Keyboard shortcuts are experimental and default off. Enable them from **Settings → Experimental → Keyboard shortcuts**. The command palette has its own **Command palette (⌘K)** gate.

| Shortcut | Action |
| --- | --- |
| ⌘S | Show or hide the sidebar when sidebar navigation is enabled |
| ⌘F | Focus search for the current page |
| ⌘; | Open Settings |
| ⌘N | Run Container |
| ⌘U | Run an image update check |
| ⌘I | Open Activity |
| ⌘K | Open the command palette when the palette is enabled |
| Esc | Close the active morph panel, sheet, or palette where supported |

Additional menu actions without a shortcut live under the menu-bar **Create**, **Navigate**, **Settings**, and **Help** menus. Experimental features hide or disable their matching menu actions until enabled.

The command palette is the fastest way to navigate or run an app/container/image/resource action: type to fuzzy-filter, use ↑/↓ to move, ↵ to run, and Esc to close.
