# Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| ⌘N | New container |
| ⌘R | Run a container |
| ⌘K | Open the command palette |
| ⌘F | Focus the search field |
| ⌘⇧R | Reload / refresh the current section |
| ⌃⌘S | Toggle the sidebar |
| ⌘1 … ⌘8 | Jump to a sidebar section (Containers, Images, Build, Volumes, Networks, Registries, System, Templates) |
| ⌘, | Settings |
| ⌘W | Close the frontmost window (Esc dismisses a sheet) |

Additional menu actions (no shortcut): File ▸ Pull Image…, File ▸ Import Compose…, Help ▸ Contained Help / Keyboard Shortcuts / Report an Issue / View Source.

The command palette (⌘K) is the fastest way to navigate or run a per-container lifecycle action — type to fuzzy-filter, ↑/↓ to move, ↵ to run.
